# Terrarium Bot
Manages a terrarium, can control heating, lights and misting all based off of min/max humidity or temperature.

## Features

- Turn off lights 5 minutes before misting and then back on again
- Turn on/off lights at set times
- Mist every 4 hours or when humidity reaches a certain level
- Turn on/off a fan when humidity reaches a certain level
- Play jungle sounds during the day/night
- Simulate sunset/sun rise by turning on lights 1 by 1
- Send alerts via Pushover if temperatures/humidity is high/low
